2) Git commands to set up a remote repository in uhura
	cd /git_repositories
	mkdir repoName
	cd ..
	chgrp git repoName -R  
	cd repoName
	git init --bare

3) Git commands to set up a local repository from an existing uhura repository
	git clone userName@uhura:/git_repositories/repoName
	git config user.email 'userName@psu.edu'
	git config user.name 'userName'
	git config --global push.default simple

4) Git commands to update from uhura and then upload local changes
	git pull					(updates repository from uhura)

	work and make local changes ..

	git add -A 					(to add all files in all subfolders)
	git fileName1 fileName2		(to add specific files)
	git commit -m 'message to explain the commit, e.g. new gains added ..'
	git push
